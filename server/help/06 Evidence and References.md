The platform supports attaching evidence from multiple sources to conditions and interventions. This ensures every treatment recommendation is backed by verifiable references.

## Evidence entries

Evidence entries are research studies or articles that support (or challenge) an intervention or condition. Each entry contains:

| Field | Description |
|-------|-------------|
| **Title** | Study or article title |
| **PubMed ID** | PMID for direct linking to PubMed |
| **DOI** | Digital Object Identifier |
| **Authors** | Author list |
| **Journal** | Publication journal |
| **Publication Date** | When the study was published |
| **Abstract** | Study abstract or summary |
| **URL** | Direct link to the full text or source |
| **Study Type** | Type of study (e.g., RCT, meta-analysis, cohort, case study) |
| **Quality Rating** | Your assessment of the study quality |
| **Sample Size** | Number of participants |
| **Summary** | Your summary of key findings relevant to this condition |

## Adding evidence from PubMed

1. Open a condition and go to the **Evidence** tab
2. Click **Search PubMed**
3. Enter search terms (e.g., "Mediterranean diet type 2 diabetes")
4. Browse results and click **Import** on relevant studies

The platform automatically fills in the title, authors, journal, abstract, PubMed ID, and DOI from the PubMed record.

## Adding evidence from ClinicalTrials.gov

1. Click **Search Clinical Trials** in the Evidence tab
2. Enter search terms
3. Browse results and import relevant trials

## Adding evidence manually

Click **Add Evidence** to create an entry manually. Fill in as many fields as possible for complete citations.

## Linking evidence to interventions

Evidence entries can be linked to:
- A **condition** (general evidence about the condition)
- An **intervention** (evidence supporting a specific treatment)
- Both (evidence about a specific treatment for a specific condition)

## Scripture references

Scripture references are Bible verses and Spirit of Prophecy quotations relevant to health principles. Each reference includes:

| Field | Description |
|-------|-------------|
| **Reference** | Citation (e.g., "3 John 1:2", "Ministry of Healing, p. 127") |
| **Text** | The quoted text |
| **Book, Chapter, Verse** | Structured Bible reference fields |
| **Translation** | Bible translation used (e.g., KJV, NKJV, ESV) |
| **Theme** | Topical theme (e.g., "health reform", "temperance") |
| **Spirit of Prophecy** | Whether this is an Ellen G. White quotation |
| **SoP Source** | Book title for Spirit of Prophecy references |
| **SoP Page** | Page number in the source book |
| **Care Domain** | Which care domain this reference relates to |

## Searching for Bible verses

1. In the condition editor, go to the **Scripture** tab
2. Click **Search Bible**
3. Enter a reference (e.g., "John 3:16") or keyword (e.g., "healing")
4. Click on a result to add it to the condition

## Searching Spirit of Prophecy

1. Click **Search EGW**
2. Enter search terms (e.g., "health reform", "temperance")
3. Browse results from Ellen G. White\u2019s writings
4. Click to add a quotation to the condition

## In-editor insert tools

When editing any document, you can also insert references directly into the text:
- Use the **Bible insert** tool to search and embed a formatted Bible quotation
- Use the **EGW insert** tool to search and embed a Spirit of Prophecy quotation
- Use the **AI explanation** tool to generate and insert an explanation of a topic

These tools are available from the editor toolbar or slash commands.
